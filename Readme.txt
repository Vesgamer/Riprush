Riprush - Descargador de videos y audio

Descripción:
Riprush es una aplicación desarrollada en Python que permite descargar videos y audio desde diferentes plataformas. Utiliza yt-dlp y ffmpeg para ofrecer una experiencia de descarga rápida y eficiente.

Características:
- Descarga de videos en diferentes calidades.
- Extracción de audio en múltiples formatos.
- Interfaz gráfica fácil de usar.
- Soporte para múltiples pistas de audio.

Requisitos:
- Windows 10 o superior.
- Python 3.8 o superior (solo para desarrollo).
- Archivos adicionales incluidos en el repositorio (ffmpeg.exe, yt-dlp.exe, etc.).

Instalación:
1. Descarga el archivo ejecutable desde la sección de "Releases".
2. Ejecuta el archivo "Riprush.exe".
3. Sigue las instrucciones en la interfaz gráfica.

Uso:
1. Pega el enlace del video que deseas descargar en el campo de texto.
2. Selecciona el tipo de descarga (video o solo audio).
3. Elige la calidad del video o las pistas de audio que deseas descargar.
4. Haz clic en "Descargar" y espera a que se complete el proceso.

Notas:
- Asegúrate de tener una conexión a Internet estable durante la descarga.
- Si el archivo es marcado como sospechoso por tu antivirus, añádelo a la lista de excepciones. Es un falso positivo.

Soporte:
Si encuentras algún problema o tienes alguna pregunta, por favor abre un "Issue" en este repositorio o contacta al desarrollador.
